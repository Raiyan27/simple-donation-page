IMPORTANT FAQs

What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects. Each element, attribute, and piece of text is represented as a node, allowing developers to manipulate the document's content, structure, and styles using languages like JavaScript. This enables dynamic interaction with web pages.

How do you select an element from the DOM?

You can select an element from the DOM using methods like:

  • document.getElementById('id')
  • document.querySelector('selector')
  • document.getElementsByClassName('class')
  • document.getElementsByTagName('tag')

These methods allow you to access elements based on their ID, CSS selectors, class names, or tag names.

What is event delegation in the context of DOM, and why is it useful?

Event delegation is attaching a single event listener to a parent element to handle events for its child elements. It's useful because it improves performance, handles dynamically added elements automatically, and simplifies code management.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using methods like element.getAttribute('attrName') to get an attribute, element.setAttribute('attrName', 'value') to set one, and element.removeAttribute('attrName') to remove it. For styles, you can access a style with element.style.property (e.g., element.style.backgroundColor), set a style with element.style.property = 'value' (e.g., element.style.color = 'red'), and remove a style by setting it to an empty string (e.g., element.style.margin = ''). These methods allow you to modify elements dynamically.
